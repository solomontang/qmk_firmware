# Solomon's KBD8x layout

| Key Combination   | Action              |
| ----------------- |:-------------------:|
| Fn + LEFT         | Prev Track          |
| Fn + RIGHT        | Next Track          |
| Fn + UP           | Volume Up           |
| Fn + DOWN         | Volume Down         |
| Fn + End          | Play/Pause          |
| Fn + Delete       | Mute                |
| Fn + W            | Up                  |
| Fn + A            | Left                |
| Fn + S            | Down                |
| Fn + D            | Right               |
| Fn + P            | Backlight pulse     |
| Fn + [            | Backlight decrement |
| Fn + ]            | Backlight increment |
| Fn + \            | Backlight toggle    |
